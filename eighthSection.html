Advanced CSS

Span & Display: block, inline, inline-block, and none

  1. Span This is a span element
    • Span elements are inline by default and do not start on a new line.
    • They can be used to style a part of the text without breaking the flow.
  2. Block: This is a block element
    • Block elements start on a new line and take up the full width available.
    • Examples include headings, paragraphs, and divs.
  3. Inline: This is an inline element
    • Inline elements do not start on a new line and only take up as much width as necessary.
    • Examples include links, spans, and images.
  4. Inline-block: This is an inline-block element
    • Inline-block elements are similar to inline elements but can have width and height.
    • They allow for more control over layout while still being inline.

  5. None: This element is not displayed
    • Elements with display: none; are not rendered at all and do not take up any space in the layout.
    • This is useful for hiding elements without removing them from the DOM.
