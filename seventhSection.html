The Cascade - Specificity and Inheritance

Least specific to most specific "Specificity"

  1. Position
    1. li { color: red; }
    2. li { color: green; }
    3. li { color: black; }
    4. li { color: white; }
  2. Specificity
    1. Element: li { color: blue; }
    2. Class: .first-class { color: red; }
    3. Attribute: li[draggable] { color: purple; }
    4. ID: #first-id { color: orange; }
  3. Type
    1. External: <link rel="stylesheet" href="./style.css">
    2. Internal: <style> { color: red; } </style>
    3. Inline: <p style="color: green;">Hello Cascade<p>
  4. Importance
    1. {color: red;}
    2. {color: green !important;}
